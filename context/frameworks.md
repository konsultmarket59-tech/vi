## Любимые фреймворки

### Для постов
- **PMHS** (Pain → More pain → Hope → Solution) — для закрытия страхов
- **До → После → Мост** — для историй трансформации
- **4P** (Picture → Promise → Proof → Push) — для продающих постов

### Для клипов
- Крючок (0–3 сек) → Демонстрация (3–20 сек) → Смысловой удар → CTA

### Для статей
- **Reader's Digest blueprint** — для контентных текстов
- **ACCA** (Attention → Comprehension → Conviction → Action) — для экспертных
