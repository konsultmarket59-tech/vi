# Контекст проекта

Эта папка содержит все данные о проекте, которые используются при генерации контентных воронок.
Файлы читаются автоматически при запуске `generate_funnel.py`.

## Структура файлов

| Файл | Содержимое |
|---|---|
| `company.md` | О компании: история, миссия, позиционирование |
| `audience.md` | Целевая аудитория: сегменты, боли, возражения |
| `pricing.md` | Цены, пакеты, условия |
| `contacts.md` | Контакты, адрес, ссылки, режим работы |
| `frameworks.md` | Копирайтерские фреймворки, голос бренда, примеры текстов |
| `offers.md` | Офферы, лид-магниты, акции |

## Как обновлять

Редактируй файлы напрямую через GitHub и делай коммит.
Изменения автоматически попадут в следующую генерацию.
