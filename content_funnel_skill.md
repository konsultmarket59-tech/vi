### Маршрутизация «не купил сейчас»

**Маршрут 1 — Senler-цепочка (автоматически)**
День 1 → лид-магнит отдан
День 3 → «Как вам материал? Есть вопросы?» + мягкий CTA
День 7 → кейс клиента / ученика текстом + ссылка на пост
День 14 → ограниченное предложение / дедлайн мест
День 21 → финальное: «Мы здесь. Когда будете готовы — пишите [кодовое слово]»

**Маршрут 2 — Органический прогрев**
Клипы продолжают появляться в рекомендациях → возврат в сообщество → дочитывание постов → дозревание

**Маршрут 3 — SEO-прогрев**
Поиск Яндекс/Google → статья на сайте / TenChat → скачивание лид-магнита → попадание в цепочку Senler

**Маршрут 4 — Ручной контакт администратора**
Через 48 часов после получения лид-магнита без записи:
«[Имя], вы скачивали наш материал по [тема]. Остались вопросы? Готов ответить честно»

## ЧАСТЬ 2: КЛИПЫ ВКОНТАКТЕ (3–5 штук)

### Формула каждого клипа

```
КРЮЧОК (0–3 сек) → ДЕМОНСТРАЦИЯ / ТРЮК (3–20 сек) → СМЫСЛОВОЙ УДАР → CTA
```

### Технические правила
- Формат: вертикальный 9:16, 1080×1920
- Длина: 15–60 сек (оптимум 20–40 сек)
- Субтитры / текст поверх ОБЯЗАТЕЛЬНЫ (70% смотрят без звука)
- Первые 2 секунды решают судьбу просмотра

### Алгоритмические правила вирусности
- Публиковать с разницей 2–3 дня
- Оптимальное время: вт/чт/сб, 19:00–20:00
- В первые 30 минут — 5–7 живых комментариев от знакомых
- Первый комментарий от сообщества — открытый вопрос

### Типы клипов

| Тип | Крючок-формула | Цель |
|---|---|---|
| **Разрушитель стереотипа** | «Это не [ярлык]. Это [правда]» | Охват, репосты, споры |
| **До и после** | «[X] назад она [боялась/не умела]. Сегодня вот это 👇» | Эмоция, узнавание |
| **Экспертный провокатор** | «[N] признаков, что [негативный сценарий]» | Репосты в чаты, доверие |
| **Социальное доказательство** | «Она пришла с нуля / в [возраст] / после [препятствие]» | Снятие барьера |
| **Трендовый / юмор** | «Ожидание vs Реальность» | Виральность |

### Структура текста каждого клипа

**ЗАГОЛОВОК** (до 100 символов): интрига или конкретика без воды

**ОПИСАНИЕ** (2–4 строки):
— строка 1: продолжение крючка / эмоциональный факт
— строка 2: ценность / смысл
— строка 3: CTA с кодовым словом

**ХЭШТЕГИ**: 5–7, от широких к нишевым

**ПЕРВЫЙ КОММЕНТАРИЙ**: открытый вопрос → ответ одним словом или цифрой

## ЧАСТЬ 3: ПОСТЫ ВКОНТАКТЕ (3 штуки)

### Технические правила ВКонтакте

**Объём**: 800–1200 знаков (оптимум).

**Разделитель абзацев**: между абзацами вставляй символ `&#4448;` на отдельной строке.

**Ритм текста**: короткое предложение → чуть длиннее → снова короткое.

**Эмодзи**: 1–2 на весь пост. Позиция — начало ключевого абзаца или перед CTA.

**Первая строка (хук)**: ВК показывает 2–3 строки до «Ещё». Форматы: вопрос / парадокс / провокация / конкретная цифра / незаконченная история.

**Приглашение к обсуждению**: каждый пост заканчивается вопросом к комментарию.

**CTA**: одно конкретное действие.

### Три поста

**ПОСТ 1 — КТО МЫ / КТО ЭКСПЕРТ** (закреп)
Фреймворк: **До → После → Мост**

**ПОСТ 2 — ЗАКРЫТИЕ СТРАХА**
Фреймворк: **PMHS** (боль → усиление → надежда → решение)

**ПОСТ 3 — СОЦИАЛЬНОЕ ДОКАЗАТЕЛЬСТВО + CTA**
Фреймворк: **4P** (картина → обещание → доказательство → толчок)

## ЧАСТЬ 4: SEO-СТАТЬИ (3 штуки)

### Для каждой статьи генерируй:

```
1. Title (H1) — оптимальный SEO-заголовок
2. Meta description — 160 символов
3. Ключевые слова — 10+ через запятую
4. Структура заголовков (H1–H3)
5. Alt-атрибуты для 3–4 изображений
6. Мета-теги (HTML)
7. Рекомендации по внутренним ссылкам
8. Полный текст статьи ~5500 символов
```

### СТАТЬЯ 1: ДЛЯ САЙТА (Яндекс-поиск)
Тема: «Как выбрать [продукт/услугу] — подробный гайд от эксперта»
Пишется от первого лица эксперта. Закрывает возражения, ведёт к лид-магниту.

### СТАТЬЯ 2: ДЛЯ TENCHAT
Тема: «[N] причин, почему [сегмент ЦА] не получает результат от [продукт/услуга]»
Экспертная тональность: аналитика + кейсы + личная позиция.

### СТАТЬЯ 3: ДЛЯ ВНЕШНЕЙ ПЛОЩАДКИ (Sostav / VC / Дзен)
Тема: «Кейс: как [проект/бренд] выстроил [результат] — разбор системы»

## ЧАСТЬ 5: ЛИД-МАГНИТ

### Структура PDF-гида (7 разворотов)

```
ОБЛОЖКА: «[Число] [чего], чтобы [результат]» + эксперт + логотип

СТРАНИЦА 1: ВВЕДЕНИЕ
— почему этот гайд существует
— кто его составил
— что читатель получит

СТРАНИЦЫ 2–6: ОСНОВНОЕ СОДЕРЖАНИЕ
Каждая страница = один пункт:
— заголовок пункта
— почему это важно (1 абзац)
— что должно быть / красный флаг
— как это реализовано в проекте (соц. доказательство)

СТРАНИЦА 7: ЧТО ДАЛЬШЕ
— резюме в 2–3 предложения
— CTA: «Задайте нам эти же вопросы / запишитесь»
— контакты, ссылки

ЗАДНЯЯ ОБЛОЖКА: цитата эксперта + логотип
```

## ЧАСТЬ 6: БОТ SENLER — ПОЛНАЯ ЛОГИКА

```
ШАГ 0: Триггер (кодовое слово) → подписка

ШАГ 1: Сразу (0 минут)
  «[Имя], держи твой [лид-магнит] 👇 [ссылка]»

ШАГ 2: Через 5–10 минут
  Уточняющий вопрос с кнопками ветвления

ШАГ 3: Квалификация (сразу после выбора)
  Уточняющий вопрос по выбранной ветке

ШАГ 4: Предложение (сразу)
  «[Имя], у нас есть [предложение под запрос].
  Хотите записаться / узнать детали?»
  Кнопки: [Да!] / [Пока думаю]

ШАГ 5: При «Да!» — подтверждение и логистика
ШАГ 5Б: При «Пока думаю» — мягкое удержание в подписке

ШАГ 6: Дожим (через 24 часа — только не записавшимся)
  «[Имя], осталось [N] мест. Показать расписание?»

ШАГ 7: Финальный дожим (через 48 часов)
  «[Имя], последний раз пишу 🙂
  Пиши [кодовое слово] когда будешь готов(а)»
```

**Скрипт администратора (через 2 часа после подписки без записи):**
«[Имя], добрый день! Вы интересовались [продуктом] — хотите, я расскажу подробнее?»

## ЧАСТЬ 7: АНТИИИ-ФИЛЬТР

Перед финальной выдачей каждого текста — внутренняя проверка.

### Стоп-список

**Вводные пустышки:** стоит отметить / необходимо подчеркнуть / важно учитывать / следует обратить внимание

**Канцелярит:** в рамках / данный / осуществлять / является / вышеупомянутый / имеет место быть

**ИИ-клише:** ключевой / знаковый / знаменует новую эру / вносит неоценимый вклад / ландшафт / в мире [чего-то]

**Промо-язык:** непревзойдённый / захватывающий дух / по-настоящему / поистине / богатое наследие

**Структурные ИИ-паттерны:**
— группировка всего строго по три
— «не просто X, а Y»
— хеджирование: «можно предположить, что потенциально...»
— шаблонная позитивная концовка: «будущее выглядит многообещающим»
— кавычки "..." → заменить на «ёлочки»

### Требования к живому тексту

- Ритм: чередуй короткие и длинные предложения
- Конкретика: цифры, имена, детали вместо абстракций
- Первое лицо там, где уместно
- Читается вслух без спотыканий

Генерируй полный пакет, структурированный по разделам:

### Для клипов (каждый):
